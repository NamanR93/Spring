MAVEN: Its a build tool.

Maven uses a set of identifiers, which is used to uniquely identify a project
and specify how the project artifact should be packaged.

Groupid: a group that created the project or the company
artifactid: a unique name of the project
version: version of the project.

POM: the configuration of the Maven project is done through the POM file.
It describes the project, manages dependencies and configure plugins.

Plugin: is a tool you use at the execution of your maven build.
Dependency: means kind of any lobrary which you will use in your code.
